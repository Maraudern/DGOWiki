# 成为居民

「居民」为服务器正式成员，至此你获得「DGOWorld」的所有权限。

?> **获取方式：** <br/>
    - 「路人」通过邀请进入服务器<br/>
    - 「路人」提交【DGO-入服申请表】且管理组审核后感到值得信任<br/>
    - 「旅客」通过在线 `20h` 获得<br/>
    - 「旅客」通过【DGO-正式成员申请表】<br/>
 **增加权限：** <br/>
    - 「DGOWorld-综合服」生存区活动权限；鸟居领地购买权限<br/>
    - 「DGOWorld-纯净服」白名单
